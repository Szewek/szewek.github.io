Opcik Privacy Policy

Modified: 2016-10-31

Definitions

  • Opcik - conversation bot, which this privacy policy applies to.
  • User - invidual, group or organization, messaging with Opcik.
  • Personal data - data that can be used to identify a person, such as real name, address, phone number, e-mail address, password, credit card number.
  • Contribution - changes and additions made by an user.

Usage and user privacy

  1. Opcik collects user messages and doesn't need any specific personal data for organizing.
  2. Collected conversations are used as conversion messages to all Users. User agrees to share when he sends a message to Opcik.
  3. Opcik answers only when User sends to it a message.
  4. When a message contains blacklisted words, a whole message is ignored.
  5. User sends a message with personal data, it will be removed by an administrator.
  6. User contributes by making approperiate answers to Opcik's questions and responding to its messages.
